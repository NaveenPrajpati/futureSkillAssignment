# Help Center API Assignment

## Instructions

1. **Clone the Repository:**

2. **Frontend:**

   - Goto `frontend` folder.
   - Run `npm i`

3. **Backend:**

   - Goto `backend` folder.
   - Run `npm i`

4. **Final Step**

   - Goto frontend folder run `npm run dev` to start both `frontend` and `backend`.
   - Make sure to add data `Mongo Uri ` in `.env` file.
